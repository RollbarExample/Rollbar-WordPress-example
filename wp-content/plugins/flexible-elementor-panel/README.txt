This is an add-on for the ever popular Elementor Page Builder. Makes Elementor widgets panel flexible, draggable, resizable and folding that more space and opportunities.

== Description ==

"Elementor Flexible Panel" makes the Elementor Widgets Panel flexible, draggable, resizable and folding that more space and opportunities.

= Features =
* Draggable Elementor panel anywhere
* Collapsible Elementor panel
* Resize Elementor panel
* Dark skin for Elementor editor
* Collapsible all widgets category
* Sortable widgets category
* Add exit button in editor footer 




== Installation ==
From within WordPress' dashboard:

1. Go to Plugins -> Add New
2. Search for "Flexible Elementor Panel"
3. Click "Install"
4. Click "Activate"

Manually via FTP:

1. Upload the folder 'flexible-elementor-panel/' into the '/wp-content/plugins/' directory
2. Activate the plugin through the *Plugins* tab in WordPress
